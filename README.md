# Help Desk Desktop - Backend

Este projeto consiste no backend de um sistema de Help Desk desenvolvido em C# (.NET), seguindo boas práticas de versionamento e colaboração no GitHub.

## Principais Tecnologias
- .NET 9.0
- ASP.NET Core
- Entity Framework Core

## Estrutura do Projeto

- `src/` - Código-fonte do backend
- `tests/` - Projetos de teste automatizados
- `docs/` - Documentação complementar

## Como rodar o projeto

```sh
cd src/HelpDesk.API
dotnet run
```
